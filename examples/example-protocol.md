# Blog Post Polish

A simple three-pass process for polishing a blog post before you hit publish.

## When to Use

After you have a solid draft and want to tighten it up. Not for structural rewrites (that's a different workflow).

## Pass 1: Clarity

Read the post start to finish. For each paragraph, ask:
- Does this paragraph earn its place? If I cut it, would the post suffer?
- Is the main point of this paragraph clear in the first sentence?
- Are there words I can cut without losing meaning?

Load the Zinsser reference if available. Apply his principles: strip clutter, use active verbs, cut qualifiers.

## Pass 2: Flow

Read the post aloud (or have the AI read it back to you). Listen for:
- Sentences that all sound the same length (vary them)
- Transitions that feel mechanical ("Furthermore," "Additionally")
- Places where the energy drops

## Pass 3: Voice

Run your voice tests (if you have them in your voice profile). Ask:
- Does this sound like me or like a robot?
- Would I say this to a friend?
- Is there a single line that makes me cringe? Cut it.

## Done

The post is ready. Publish it.
